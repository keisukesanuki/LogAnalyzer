# LogAnalyzer

## これは何？

WEB サーバのアクセスログを解析する需要があるのか良くわからないツールです。  
※ Apache / Nginx に対応  

## Installation

```
git clone https://github.com/keisukesanuki/LogAnalyzer.git
chmod 777 LogAnalyzer/upload
```

## Usage
一番左の input フォーム(灰色)は、ログフォーマットのリクエスト時刻を変更しているログに対しての調整用途にご利用ください。
